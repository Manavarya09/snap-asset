snap-asset

Screenshot automation for developers. URL, element, or component — tell it what to capture, it handles the rest.

Get Started Features
# Screenshot any URL snap-asset https://myapp.com --name hero # Capture a specific element snap-asset https://myapp.com --selector ".card" # Extract entire site snap-asset extract https://site.com # Isolate a component snap-asset component ./src/Hero.tsx

What it does

Capture anything

[1]

Element Capture

Target specific elements with CSS selectors. Capture just what matters.

[2]

Site Extraction

Scrape entire websites: sections, images, components — everything at once.

[3]

Component Isolation

Render React, Vue, Svelte components in isolation with transparent backgrounds.

[4]

Auto-Optimize

PNG + WebP generated simultaneously. Retina ready at 2x scale.

[5]

Batch Mode

Define all captures in config, run once. Efficiency for the win.

[6]

Dark Mode

Capture in light or dark color scheme. Your screenshots, your style.

How it works

Three steps

1

Command

Tell snap-asset what to capture — URL, element selector, or component path.

→
2

Capture

Playwright takes the screenshot, Sharp optimizes to PNG + WebP.

→
3

Save

Auto-saves to public/, assets/, or your specified output directory.

Usage

Commands

# Capture a URL snap-asset https://myapp.com --name hero # Capture a specific element snap-asset https://myapp.com --selector ".card" --name feature # Extract entire website snap-asset extract https://competitor.com --out public/ # Capture component in isolation snap-asset component ./src/Hero.tsx --name hero # Batch capture snap-asset batch # Dark mode variant snap-asset https://myapp.com --dark --name hero-dark
